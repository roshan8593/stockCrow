# 🚀 Stock Crow

Stock Crow is a full-stack stock trading simulation platform built using modern web technologies.  
It allows users to create an account, securely log in, and manage their stock holdings, positions, and orders through an interactive dashboard.

---

## 🌐 Live Architecture

- Main Website (Landing + Signup)
- Trading Dashboard
- Backend API Server
- MongoDB Database

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- React Router
- Axios
- Material UI

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt (Password Hashing)
- Cookie-based Authentication

---

## 🔐 Features

- User Signup & Login (JWT based authentication)
- Secure password hashing with bcrypt
- Cookie-based session handling
- Logout functionality
- Dashboard with:
  - Holdings
  - Positions
  - Orders
- Cross-origin authentication handling
- RESTful API architecture
